# Description du projet

# Marvel Quiz

Une application web interactive développée avec React.js et Firebase qui permet aux utilisateurs de tester leurs connaissances sur l’univers Marvel à travers des quiz dynamiques générés à partir de la Marvel Developer API.

L’application propose une expérience ludique et immersive avec un système d’authentification sécurisé, la sauvegarde des scores, un classement des joueurs et une interface moderne entièrement responsive.

Ce projet a pour objectif de mettre en pratique la consommation d’API REST, la gestion d’état avec React, l’authentification et la persistance des données avec Firebase, ainsi que la conception d’interfaces utilisateur performantes.
